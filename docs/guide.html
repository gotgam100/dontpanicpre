01

로그인 방법

Don't Panic Pre는 Google 계정으로 로그인합니다. 별도 회원가입 없이 Google 계정만 있으면 바로 사용할 수 있습니다.

1
dontpanicpre.web.app에 접속하면 로그인 화면이 나타납니다.
2
Google로 계속하기 버튼을 클릭해 Google 계정으로 인증합니다.
3
로그인이 완료되면 홈 화면(프로젝트 선택 화면)으로 이동합니다.

같은 Google 계정으로 여러 기기에서 동시에 접속할 수 있으며, 데이터는 실시간으로 동기화됩니다.

02

홈 화면 · 프로젝트 관리

로그인 후 표시되는 화면에서 프로젝트를 생성하거나 불러옵니다.

✨
새 프로젝트
프로젝트 이름을 입력하고 생성하면 빈 스크립트가 열립니다.
📂
프로젝트 선택
이전에 작업한 프로젝트 목록에서 클릭해 바로 열 수 있습니다.
📤
수동 불러오기
내보낸 JSON 파일을 드래그하거나 선택해 프로젝트를 복원합니다.

프로젝트 데이터는 Google Firebase에 저장되며, 로그아웃 후 재로그인해도 그대로 유지됩니다.

03

메인 창 · 사이드바 메뉴

메인 화면 왼쪽 사이드바에서 프로젝트의 전체 씬 목록과 주요 기능에 접근할 수 있습니다.

🎬
씬 목록
등록된 씬이 순서대로 나열됩니다. 클릭하면 해당 씬으로 스크롤합니다.
👤
인물 목록
스크립트에서 감지된 인물 목록을 확인하고, 클릭해 인물설정으로 이동합니다.
🔍
찾기 / 바꾸기
스크립트 내 텍스트를 검색하거나 일괄 치환합니다.
⚙️
프로젝트 설정
프로젝트 이름, 씬 번호 스타일, 내보내기 등 설정을 관리합니다.
💬
채팅
프로젝트에 초대된 팀원들과 실시간 채팅을 할 수 있습니다.
📤
내보내기
스크립트나 씬브레이크다운을 PDF·JSON 등으로 내보냅니다.
04

스크립트 & 씬브레이크다운

메인 탭의 핵심 기능입니다. 스크립트를 직접 입력·편집하고, 씬별 구성요소를 정리할 수 있습니다.

스크립트 작성
→
상단 툴바에서 씬 헤딩 / 지문 / 대사 / 인써트씬 등 문단 타입을 선택하고 입력합니다.
→
씬 헤딩을 입력하면 자동으로 씬이 생성되고 사이드바 씬 목록에 표시됩니다.
→
텍스트를 드래그 후 우클릭하면 인물·소품·의상 등을 씬 요소로 등록할 수 있습니다.
씬브레이크다운

상단 탭에서 [씬브레이크다운]을 선택하면 각 씬별로 등장인물, 소품(세트·캐릭터), 의상·분장, VFX, 기타 요소를 카드 형태로 관리할 수 있습니다.

👤 등장인물
🧳 소품
👗 의상 · 분장
✨ VFX
📝 씬 메모

등록된 요소는 리스트 탭에서 항목별로 통합 관리됩니다. 삭제·수정도 리스트와 씬브레이크다운 양쪽에 실시간으로 반영됩니다.

05

리스트 · 인물설정 · 스케줄 · 제작진

리스트

씬브레이크다운에서 등록한 요소들을 한 곳에서 관리합니다. 소품 리스트, 의상·분장 리스트를 항목별로 확인하고, 메모·담당자 등을 추가할 수 있습니다.

인물설정

스크립트에서 자동 감지된 인물 목록을 관리합니다. 각 인물마다 아래 내용을 설정할 수 있습니다.

성별 · 나이
인물 색상
캐릭터 설정
등장씬 확인
메모
스케줄

촬영 일정을 관리합니다. 날짜별로 씬을 배치하고, 콜시트(Call Sheet)를 생성해 팀원들과 공유할 수 있습니다.

제작진

스태프 리스트와 배우 리스트를 관리합니다.

🎬
스태프 리스트
부서 · 역할 · 세부역할 · 이름 · 연락처 · 이메일 · 메모를 입력합니다.
🎭
배우 리스트
배역명 · 배우명 · 소속사 · 연락처1 · 연락처2 · 메모를 입력합니다.
06

채팅

프로젝트에 참여한 팀원들과 실시간 채팅을 할 수 있습니다.

→
사이드바의 채팅 버튼을 클릭해 채팅 패널을 엽니다.
→
같은 프로젝트에 접속한 팀원들과 실시간으로 메시지를 주고받을 수 있습니다.
→
메시지는 Firebase에 저장되어 나중에 접속해도 이전 대화를 확인할 수 있습니다.

프로젝트 설정에서 팀원을 초대하면 함께 스크립트를 실시간 공동 편집하고 채팅할 수 있습니다.
