Beta 1.0 First Release 2026년 6월 1일
New — 새 기능
  • 스크립트 에디터 — 씬 헤딩, 지문, 인물명, 대사를 전용 유형으로 작성. 입력 시 자동 형식화 및 씬 번호 자동 부여.
  • 인서트씬 — 씬 내 구간을 인서트씬으로 지정. INT/EXT, 시간대, 장소명 개별 설정 지원.
  • 씬리스트 — 스크립트에서 씬 목록을 자동 생성. 의상, 분장, 소품, 인물소품, 특수효과, 기타 요소 관리.
  • 장소리스트 — 씬 헤딩 장소를 자동 수집. 장소 설명·실제 촬영지 기입 및 드래그 순서 변경.
  • 소품리스트 — 씬 브레이크다운과 연동되는 소품 통합 관리.
  • 의상리스트 — 인물별 의상 항목 관리. 씬 브레이크다운과 연동.
  • 촬영 스케줄 — 씬을 달력에 드래그하여 날짜별 배치. 순서 변경 시 일촬표에 자동 반영.
  • 일촬표(Call Sheet) — 촬영 스케줄 기반 일일촬영계획표 자동 생성. PDF 출력 지원.
  • 팀 협업 — Google 계정 로그인, 초대 코드 기반 프로젝트 공유. 소유자·참여자·열람자 권한 구분.
  • 클라우드 저장 — Firebase Firestore 기반 실시간 저장 및 버전 히스토리(10회 자동 백업).
  • 다크/라이트 테마 — 헤더 토글로 테마 즉시 전환. 랜딩 페이지는 항상 다크 유지.
  • PDF 출력 — 씬리스트, 장소리스트, 소품리스트, 의상리스트 PDF 인쇄 지원.
  • PWA 지원 — 홈 화면에 추가하여 앱처럼 사용 가능. 오프라인 캐시 지원.
Improve — 개선
  • 시간대 6종 통일 — 새벽·오전·오후·해질녘·저녁·밤으로 표기 일원화. '미정' 옵션 추가.
  • 사이드바 — 접기/펼치기 지원. 씬 목록 클릭 시 스크립트 해당 위치로 이동.
  • 씬리스트 D/N·I/E — 읽기 전용으로 변경. 씬 헤딩 또는 인서트씬 모달에서만 수정 가능.
