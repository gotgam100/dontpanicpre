Don't Panic Pre

보안

Don't Panic Pre는 제작 자료와 협업 정보를 안전하게 다루기 위해 인증, 권한, 저장소 보안 원칙을 적용합니다.

1. 인증

로그인은 Firebase Auth 기반 Google 인증을 사용합니다. 비밀번호는 Don't Panic Pre 서버에 직접 저장하지 않습니다.

2. 권한 관리

  • 프로젝트는 소유자, 참여자, 열람자 권한으로 구분됩니다.
  • 초대 코드 또는 이메일 기반 권한에 따라 프로젝트 접근이 제한됩니다.
  • 열람자는 편집 권한 없이 프로젝트를 확인하는 용도로 제한됩니다.

3. 데이터 저장

프로젝트 데이터는 Firestore에 저장되며, 전송 구간은 HTTPS를 통해 보호됩니다. 실시간 협업 데이터는 프로젝트 단위로 동기화됩니다.

4. 사용자 권장 사항

  • 초대 코드를 공개 채널에 공유하지 마세요.
  • 프로젝트 참여자 목록을 정기적으로 확인하세요.
  • 공용 기기에서는 작업 후 반드시 로그아웃하세요.
