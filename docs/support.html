자주 묻는 질문

Q. 다른 사람을 프로젝트에 초대하는 방법이 궁금합니다.

프로젝트 설정 화면에서 참여자 초대 코드를 생성하거나, 이메일로 직접 초대할 수 있습니다. 참여자는 편집 권한, 열람자는 읽기 전용 권한으로 초대할 수 있습니다.

Q. 프로젝트를 백업하고 싶습니다.

프로젝트 설정에서 'JSON 내보내기'를 통해 전체 프로젝트 데이터를 파일로 저장할 수 있습니다. 중요한 작업 전에 정기적으로 백업하시길 권장합니다.

Q. 프로젝트 데이터가 사라졌습니다.

Firebase 실시간 동기화를 사용하므로 인터넷이 연결된 상태에서 재접속 시 자동으로 복원됩니다. 문제가 지속될 경우 이메일로 문의해주세요.

Q. 계정을 삭제하고 싶습니다.

계정 삭제를 원하시면 yogotgam@gmail.com으로 요청해주시면 처리해드립니다. 삭제 요청 후 30일 이내 모든 데이터가 영구 삭제됩니다.
