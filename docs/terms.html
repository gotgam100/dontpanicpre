Don't Panic Pre

이용약관

Don't Panic Pre를 이용하기 전에 아래 약관을 확인해주세요. 서비스를 사용하는 경우 본 약관에 동의한 것으로 봅니다.

1. 서비스 목적

Don't Panic Pre는 영화, 드라마, 영상 제작의 프리 프로덕션 과정을 돕기 위한 대본 편집, 씬 브레이크다운, 촬영 스케줄, 일촬표 관리 도구입니다.

2. 사용자 책임

  • 사용자는 본인이 업로드하거나 작성한 콘텐츠에 필요한 권리를 보유해야 합니다.
  • 타인의 권리, 개인정보, 영업비밀을 침해하는 자료를 입력하거나 공유해서는 안 됩니다.
  • 프로젝트 초대 코드와 계정 접근 권한은 사용자가 안전하게 관리해야 합니다.
  • 중요한 자료가 담긴 프로젝트는 반드시 프로젝트 내보내기를 통해 json파일로 백업하길 권장합니다.

3. 콘텐츠와 프로젝트 데이터

사용자가 작성한 대본과 프로젝트 데이터의 권리는 사용자에게 있습니다. Don't Panic Pre는 서비스 제공, 저장, 동기화, 백업, 오류 대응에 필요한 범위에서 해당 데이터를 처리합니다.

4. 서비스 변경 및 중단

기능 개선, 보안 대응, 시스템 점검 등의 사유로 서비스 일부가 변경되거나 일시 중단될 수 있습니다. 중요한 변경이 있을 경우 가능한 범위에서 사전에 안내합니다.

5. 면책

Don't Panic Pre는 안정적인 서비스를 위해 노력하지만, 네트워크 장애, 외부 서비스 장애, 사용자의 오조작 등으로 발생한 손해에 대해 법령이 허용하는 범위 내에서 책임을 제한합니다.
